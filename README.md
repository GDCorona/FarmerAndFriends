# 🌾 Farmer and Friends

It all began from a small riddle given by my college teacher. A sudden impulse propelled me to solve it with Python, all the way to developing my first ever indie game with Pygame! Props to my teacher for tangentially inspiring me to explore how to turn a mere script into a fully playable game - a conundrum that has stuck in my mind for years. Might consider refining this project and transforming more of my old code files into actual minigames like this so stay tuned!

![Gameplay Screenshot](images/icon.png) 

## 🎮 About the Game
A farmer from Morioh-chou is stranded at the riverbank with a bag of beans and his two pets: a goose and a fox. Help him bring everything safely to the other side of the river without breaking the constraints. 

## 🚀 Download and Play (Windows)
1. Go to the [Releases](../../releases) tab on the right side of this page.
2. Download the latest `FarmerAndFriends.zip` file.
3. Extract the folder to your computer.
4. Double-click `FarmerAndFriends.exe` and enjoy!

## 💻 For Developers (Run from Source)
**Prerequisites:**
* Python 3.x installed
* Pygame library

**Installation:**
1. Clone this repository:
   ```bash
   git clone [https://github.com/GDCorona/FarmerAndFriends.git]
2. Navigate to the project folder:
   ```bash
   cd FarmerAndFriends
3. Install the required libraries:
   ```bash
   pip install -r requirements.txt
4. Run the game:
   ```bash
   python FarmerAndFriends.py
